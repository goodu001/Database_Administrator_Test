หน้าที่และความรับผิดชอบ

มีประสบการณ์เกี่ยวกับการออกแบบหรือพัฒนาระบบฐานข้อมูลขององค์กรธุรกิจภาครัฐหรือเอกชน
กำหนดสิทธิ์การใช้งานฐานข้อมูลของแต่ละผู้ใช้งาน ตรวจสอบการเข้าใช้ฐานข้อมูลของผู้ใช้ รวมไปถึงออกแบบและสร้างฐานข้อมูลเพื่อรองรับกระบวนการทำงานของ สสส.ได้
รักษาป้องกันความปลอดภัยของข้อมูลให้เป็นไปตามมาตรฐาน และกำหนดนโยบายในการสำรองและกู้คืนข้อมูล รวมทั้งวางแผนการทดสอบ
ปรับแต่งฐานข้อมูลและติดตามการใช้งานฐานข้อมูลอย่างสม่ำเสมอ เพื่อตรวจสอบและปรับปรุงประสิทธิภาพในการใช้งานฐานข้อมูล
บริหารข้อมูลบุคคลและองค์กร ให้เป็นปัจจุบันและทำงานร่วมกันกับผู้เกี่ยวข้องทั้ง สสส. และบริษัทผู้พัฒนาระบบเพื่อทำให้การใช้ฐานข้อมูลมีประสิทธิภาพ
มีความเข้าใจเกี่ยวกับการออกแบบและจัดทำ Data Cleaning / Data Analysis รวมไปถึงจัดทำแผนรองรับการเพิ่มขึ้นของข้อมูลที่จัดเก็บ
ดูแลและรับผิดชอบบริหารจัดการระบบเทคโนโลยีสารสนเทศที่ได้รับมอบหมายให้สามารถใช้งานได้อย่างราบรื่น
ดูแลและบริหารจัดการงานประชุม รวมไปถึงการเจรจาความร่วมมือในเรื่องต่างๆ ให้เป็นไปอย่างมีประสิทธิภาพ
สร้างภาพลักษณ์ที่ดีและสร้างความน่าเชื่อถือ ให้กับผู้ใช้งานระบบฯ และผู้เกี่ยวข้อง 
คุณสมบัติ

วุฒิการศึกษาไม่ต่ำกว่าปริญญาตรีหรือเทียบเท่า ทางด้านวิทยาการคอมพิวเตอร์ วิศวกรรมคอมพิวเตอร์ วิทยาศาสตร์คณิตศาสตร์ วิทยาศาสตร์สถิติ หรือสาขาอื่นที่เกี่ยวข้อง
อายุไม่เกิน 40 ปี
มีประสบการณ์ในการทำงานไม่น้อยกว่า 3 ปี และต้องเป็นประสบการณ์ในการดูแลระบบฐานข้อมูลหรืองานที่เกี่ยวข้อง
มีความรู้ทางด้าน Database และสามารถเขียนคำสั่ง SQL ในการดึงข้อมูลได้และมีความรู้ทางด้าน Data Relational
มีความเชี่ยวชาญทางด้านฐานข้อมูล Microsoft SQL Server และสามารถเขียน Store Procedure ได้
สามารถใช้ AI ในการปฏิบัติงานเบื้องต้นได้
มีความสามารถในการศึกษาหาข้อมูล วิเคราะห์ข้อมูล สรุปผล และนำเสนอข้อมูลให้ที่ประชุมเข้าใจง่ายได้
มีมนุษยสัมพันธ์ที่ดี สามารถติดต่อสื่อสาร ประสานงาน และทำงานร่วมกับผู้อื่นได้ดี
มีใจรักในการบริการ ดูแลและช่วยเหลือผู้ใช้งาน ให้ได้รับความพึงพอใจจากการทำงาน
